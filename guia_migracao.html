Guia de Migração de Computador

Finanças Fácil - Sistema de Controle Financeiro

Visão Geral

Este guia orienta o processo de migração e inicialização do projeto Finanças Fácil para um novo computador. O sistema possui um frontend (React/Vite) e um backend (Node.js/Express) integrado com banco de dados PostgreSQL hospedado no Supabase (nuvem).

Passos para Funcionamento

1
Instalar o Node.js

O novo computador precisa possuir o ambiente de execução JavaScript Node.js instalado.

👉 Acesse nodejs.org, baixe e instale a versão recente LTS recomendada.

2
Preparar Arquivos de Configuração (.env)

Garanta que as variáveis de ambiente com segredos estejam presentes. Se não existirem, copie e preencha a partir dos arquivos de exemplo.

Configuração do Backend (server/.env):

PORT=5000
NODE_ENV=development
JWT_SECRET=sua_chave_secreta_aqui
JWT_EXPIRE=7d

# Envio de E-mail (Opcional)
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=seu_email@gmail.com
EMAIL_PASS=sua_senha_de_app_do_gmail

Configuração do Frontend (.env na raiz):

VITE_API_URL=http://localhost:5000/api
3
Instalar Dependências (node_modules)

As pastas node_modules antigas devem ser apagadas antes da cópia do projeto para evitar conflito de sistemas operacionais.

No Windows (Recomendado): execute o script automático clicando duas vezes em:

setup.bat (localizado na raiz do projeto)

No terminal (Manual):

# Instalar dependências do Frontend (raiz)
npm install

# Instalar dependências do Backend (pasta server)
cd server
npm install
4
Iniciar o Sistema

Com as dependências instaladas e arquivos de configuração criados, inicialize os servidores.

No Windows (Recomendado): execute o script de inicialização dupla clicando em:

start-tudo.bat (localizado na raiz do projeto)

No terminal (Manual): execute em dois terminais separados:

# Terminal 1 - Iniciar Backend
cd server
npm run dev

# Terminal 2 - Iniciar Frontend
npm run dev
⚠️ Nota Sobre os Dados

Como o banco de dados é hospedado remotamente (Supabase na Nuvem), todas as transações, orçamentos e cadastros existentes estarão acessíveis imediatamente no novo computador assim que o login for realizado com sucesso.
