## 📋 O QUE VOCÊ VAI VER AGORA

### Quando o BACKEND não estiver rodando:

```
╔══════════════════════════════════════════════╗
║ 🔴 Backend não está respondendo              ║
║                                              ║
║ O servidor backend não está rodando ou       ║
║ não está acessível                           ║
║                                              ║
║ 📋 Como resolver:                            ║
║ 1. Abra um NOVO terminal                     ║
║ 2. Execute: cd server                        ║
║ 3. Execute: npm install                      ║
║ 4. Execute: npm run dev                      ║
║ 5. Aguarde ver: "🚀 Servidor rodando..."     ║
║ 6. Tente fazer login novamente               ║
║                                              ║
║ 🌐 URL da API: http://localhost:5000/api    ║
║ 📊 Status: Backend não respondendo           ║
╚══════════════════════════════════════════════╝
```

## 🚀 COMO TESTAR AGORA

### 1. Recarregue a página

```
Pressione Ctrl + Shift + R
```

### 2. Tente fazer login ou cadastro

```
http://localhost:5173/login
ou
http://localhost:5173/cadastro
```

### 3. Veja a mensagem de erro detalhada

A tela vai mostrar **EXATAMENTE** o que está errado e **COMO RESOLVER**!

## 📸 EXEMPLOS DE MENSAGENS

### Erro 1: Backend não está rodando
```
🔴 Backend não está respondendo

Problema: O servidor backend não está rodando

Como resolver:
1. Abra um NOVO terminal
2. cd server
3. npm run dev
4. Aguarde ver: "🚀 Servidor rodando..."

URL da API: http://localhost:5000/api
Status: Backend não respondendo
```

### Erro 2: E-mail ou senha incorretos
```
❌ E-mail ou senha incorretos

Verifique suas credenciais e tente novamente.
```

### Erro 3: Campos vazios
```
⚠️ Preencha todos os campos

Nome e e-mail são obrigatórios
```
